about=PromptMap — The PromptMapping Plugin. Turns your current QGIS canvas into a styled, georeferenced raster layer using generative AI. Render the live map view, write a prompt, and receive a GeoTIFF back — ready to use as a new layer. Supports FLUX.1 Kontext, FLUX 1.1 Ultra, FLUX.2 Editing (Black Forest Labs) and Gemini 3 Pro Image (Google). Each result is georeferenced and documented with a GeoPackage containing model, prompt, timestamp and extent.
